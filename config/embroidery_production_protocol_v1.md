# Production Protocol: Automated & Manual Digitizing Specifications

Target output: machine-ready embroidery stitch files (.DST, .PES) after approval.

Every incoming customer submission requires two planned production variants:
- Standard: max 2.5 in x 2.5 in
- Expanded: max 6.0 in x 6.0 in

Rules:
- Preserve aspect ratio.
- No stretching, squeezing, or warping.
- Scale uniformly so longest dimension fits inside target box.
- Both width and height must stay under the target threshold.
- Use clean satin-column planning for hand-drawn line art where feasible.
- Do not mark DST/PES generated unless real files exist.
- Do not halt production planning just because customer omitted dimensions.
